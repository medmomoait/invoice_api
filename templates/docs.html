📄 Invoice API Documentation

Welcome to the Invoice API! This API allows you to generate professional PDF invoices with minimal input. Simply POST your invoice data and get back a downloadable PDF file.

🔐 Get Your API Key

Visit this payment link to pay $1 and receive your API key.

📥 Endpoint

POST /generate-invoice

📤 Request Headers

  • Authorization: Bearer YOUR_API_KEY
  • Content-Type: application/json

🧾 Request Body

{
  "company_name": "Tech Co.",
  "client_name": "John Doe",
  "invoice_number": "INV-001",
  "items": [
    {"description": "Website Design", "quantity": 1, "unit_price": 500},
    {"description": "Hosting (12 months)", "quantity": 1, "unit_price": 120}
  ]
}

📄 Response

{
  "pdf_url": "https://your-domain.com/static/invoices/INV-001.pdf"
}
